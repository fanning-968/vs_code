## 📚 Vim 模式说明

- **Normal 模式**：默认模式，用于导航和命令
- **Insert 模式**：输入文本（按 `i`, `a`, `o` 等进入）
- **Visual 模式**：选择文本（按 `v`, `V`, `Ctrl+v` 进入）
- **Command 模式**：执行命令（按 `:` 进入）

---

## 🧭 1. 光标移动（Navigation）

| 按键         | 功能                                           |
| ------------ | ---------------------------------------------- |
| `h/j/k/l`  | 左/下/上/右移动                                |
| `w`        | 跳到下一个单词开头                             |
| `b`        | 跳到上一个单词开头                             |
| `e`        | 跳到当前/下一个单词末尾                        |
| `$`        | 跳到行尾                                       |
| `^`        | 跳到行首（非空字符）                           |
| `0`        | 跳到行首（绝对开头）                           |
| `gg`       | 跳到文件开头                                   |
| `G`        | 跳到文件结尾                                   |
| `数字 + G` | 跳到第 N 行（如 `5G` 跳到第 5 行）           |
| `Ctrl+u`   | 向上翻半页                                     |
| `Ctrl+d`   | 向下翻半页                                     |
| `Ctrl+f`   | 向下翻一页                                     |
| `Ctrl+b`   | 向上翻一页                                     |
| `f + 字符` | 跳到当前行下一个指定字符（如 `f,` 跳到逗号） |
| `F + 字符` | 跳到当前行上一个指定字符                       |
| `t + 字符` | 跳到当前行下一个指定字符前一个位置             |
| `;`        | 重复上一次 `f`/`F`/`t` 操作              |
| `,`        | 反向重复上一次 `f`/`F`/`t` 操作          |

## ✏️ 2. 编辑操作（Editing）

| 按键          | 功能                                                                    |
| ------------- | ----------------------------------------------------------------------- |
| `i`         | 在光标前进入插入模式                                                    |
| `a`         | 在光标后进入插入模式                                                    |
| `o`         | 在下方新建一行并进入插入模式                                            |
| `O`         | 在上方新建一行并进入插入模式                                            |
| `x`         | 删除当前字符                                                            |
| `dd`        | 删除当前行                                                              |
| `数字 + dd` | 删除 N 行（如 `3dd` 删除 3 行）                                       |
| `d + 动作`  | 删除指定范围（如 `dw` 删除单词，`d$` 删除到行尾）                   |
| `yy`        | 复制当前行                                                              |
| `数字 + yy` | 复制 N 行                                                               |
| `y + 动作`  | 复制指定范围（如 `yw` 复制单词，`y$` 复制到行尾）                   |
| `p`         | 在光标后粘贴                                                            |
| `P`         | 在光标前粘贴                                                            |
| `u`         | 撤销                                                                    |
| `Ctrl+r`    | 重做                                                                    |
| `r + 字符`  | 替换当前字符（如 `rx` 将当前字符改为 x）                              |
| `R`         | 进入替换模式（逐个替换）                                                |
| `c + 动作`  | 删除并进入插入模式（如 `cw` 删除单词并编辑，`c$` 删除到行尾并编辑） |
| `s`         | 删除当前字符并进入插入模式                                              |
| `S`         | 删除当前行并进入插入模式                                                |

## 🔍 3. 搜索与替换（Search & Replace）

| 按键                           | 功能                     |
| ------------------------------ | ------------------------ |
| `/` + `关键词` + `Enter` | 向下搜索关键词           |
| `?` + `关键词` + `Enter` | 向上搜索关键词           |
| `n`                          | 跳到下一个搜索结果       |
| `N`                          | 跳到上一个搜索结果       |
| `*`                          | 搜索光标下的单词（向下） |
| `#`                          | 搜索光标下的单词（向上） |
| `:s/old/new/g`               | 替换当前行所有匹配项     |
| `:%s/old/new/g`              | 替换整个文件所有匹配项   |
| `:%s/old/new/gc`             | 替换时逐个确认           |

## 🧪 8. 实用技巧

- **重复操作**：`数字 + 操作`（如 `5j` 向下移动 5 行，`3x` 删除 3 个字符）
- **撤销/重做**：`u` / `Ctrl+r`
- **多光标编辑**：`Ctrl+D` 选中相同词，`Ctrl+U` 取消光标，`Shift+Alt+I` 在每行末尾插入光标
- **快速跳转**：`gd`（跳转到定义）、`gi`（跳转到上次编辑位置并进入插入模式）
- **行操作**：`>>`（向右缩进）、`<<`（向左缩进）、`==`（自动缩进当前行）

## 🧩 9. 模式切换

| 按键                          | 功能                   |
| ----------------------------- | ---------------------- |
| `Esc` 或 `jj`（你配置的） | 从插入模式回到正常模式 |
| `i/a/o`                     | 从正常模式进入插入模式 |
| `v/V/Ctrl+v`                | 进入视觉模式           |

## 🎯 使用建议

1. **先熟悉基础**：`hjkl`、`i`、`Esc`、`dd`、`yy`、`p`
2. **练习组合操作**：`d2w`（删除两个单词）、`yG`（复制到文件末尾）
3. **利用数字前缀**：`3j`、`5x`、`2dd` 等
4. **多光标编辑**：`Ctrl+D` 是 VS Code 中比原生 Vim 更强大的功能
5. **自定义快捷键**：用空格键前缀（`<space>`）的命令提升效率
